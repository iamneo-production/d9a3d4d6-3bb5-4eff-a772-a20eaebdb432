## Run Locally
If you are in WORKSPACE directory, go to /backend

`cd backend`

In the backend directory, run the following command to install requirements and 
setup flask server.

```bash
  `sh local_setup.sh`
  `sh local_run.sh`
```
In the fronted directory, run

`npm run serve`
